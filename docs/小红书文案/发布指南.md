# 小红书发布指南

## 发布规则

1. **图片比例**：全部使用 3:4 竖图
2. **发布节奏**：每天 1 篇，共 30 天
3. **禁止内容**：
   - ❌ 不能带任何站外链接
   - ❌ 不能带联系方式
   - ❌ 不能提到其他平台名称（如微信、支付宝等）
4. **可以出现**：产品名称「醒美闪图」

## 配图建议

- 真实效果展示
- 前后对比图
- 操作界面截图
- 多场景成品展示
