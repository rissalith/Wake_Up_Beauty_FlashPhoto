# 醒美闪图后台管理系统 软件说明书

## 一、软件基本信息

- **软件全称**：醒美闪图后台管理系统软件
- **软件简称**：醒美闪图管理系统
- **版本号**：V1.0
- **开发完成日期**：2026年1月
- **软件分类**：应用软件/管理信息系统
- **运行平台**：Web浏览器（Chrome/Firefox/Edge）
- **开发语言**：JavaScript、Vue.js
- **软件规模**：约12000行代码

## 二、软件概述

### 2.1 软件背景

随着醒美闪图AI图片生成小程序用户量的增长，需要一套完善的后台管理系统来支撑日常运营工作。本系统为运营人员提供用户管理、订单管理、内容管理、数据统计等功能，实现对小程序业务的全面管控。

### 2.2 软件目的

本软件的主要目的是：
1. 提供可视化的运营管理界面
2. 实现用户数据的查询和管理
3. 支持场景配置的动态调整
4. 提供业务数据的统计分析

### 2.3 软件功能概述

本软件主要包含以下功能模块：
- 管理员认证模块
- 仪表盘统计模块
- 用户管理模块
- 订单管理模块
- 照片管理模块
- 场景配置模块
- 素材管理模块
- 系统监控模块

## 四、功能详细说明

### 4.1 管理员认证模块

#### 4.1.1 登录功能

管理员通过用户名和密码登录系统，系统验证身份后生成JWT令牌。

**主要功能**：
- 用户名密码登录
- 登录状态保持
- 自动登出机制
- 密码加密存储

#### 4.1.2 权限管理

系统支持多级管理员权限，不同角色拥有不同的操作权限。

**权限级别**：
- 超级管理员：全部权限
- 运营管理员：用户、订单、内容管理
- 数据分析员：只读统计数据

### 4.2 仪表盘统计模块

#### 4.2.1 核心指标展示

在首页仪表盘展示关键业务指标：

| 指标名称 | 说明 |
|---------|------|
| 今日新增用户 | 当日注册用户数 |
| 今日活跃用户 | 当日登录用户数 |
| 今日生成照片 | 当日AI生成照片数 |
| 今日收入 | 当日充值金额 |
| 累计用户数 | 总注册用户数 |
| 累计照片数 | 总生成照片数 |

#### 4.2.2 趋势图表

提供多维度的数据趋势分析：
- 用户增长趋势（日/周/月）
- 收入趋势图
- 照片生成量趋势
- 用户活跃度分析

### 4.3 用户管理模块

#### 4.3.1 用户列表

展示所有注册用户的信息列表，支持多条件筛选和搜索。

**展示字段**：
- 用户ID
- 昵称
- 头像
- 注册时间
- 最后登录时间
- 积分余额
- 生成照片数
- 状态

#### 4.3.2 用户详情

查看单个用户的详细信息：
- 基本信息
- 积分变动记录
- 照片生成历史
- 充值订单记录
- 邀请记录

#### 4.3.3 用户操作

支持对用户进行管理操作：
- 积分调整（增加/扣减）
- 账号状态变更（启用/禁用）
- 用户信息编辑

### 4.4 订单管理模块

#### 4.4.1 订单列表

展示所有充值订单，支持按状态、时间筛选。

**订单状态**：
- 待支付
- 已支付
- 已取消
- 已退款

#### 4.4.2 订单详情

查看订单详细信息：
- 订单号
- 用户信息
- 充值金额
- 获得积分
- 支付方式
- 支付时间
- 订单状态

### 4.5 照片管理模块

#### 4.5.1 照片列表

展示所有生成的照片记录：
- 照片ID
- 用户信息
- 使用场景
- 原图/结果图
- 生成时间
- 消费积分

#### 4.5.2 照片审核

对生成的照片进行内容审核：
- 查看原图和结果图
- 标记违规内容
- 删除违规照片

### 4.6 场景配置模块

#### 4.6.1 场景列表

管理所有AI生成场景：
- 场景名称
- 场景图标
- 场景状态（上线/下线）
- 消费积分
- 排序权重

#### 4.6.2 场景编辑

编辑场景的详细配置：
- 基本信息（名称、描述、图标）
- 步骤配置（动态添加/删除步骤）
- 选项配置（每个步骤的可选项）
- Prompt模板配置
- 多语言配置

#### 4.6.3 步骤管理

配置场景的交互步骤：
- 步骤类型（单选、多选、滑块、图片上传）
- 步骤选项
- 依赖关系
- 显示条件

### 4.7 素材管理模块

#### 4.7.1 素材库

管理系统使用的图片素材：
- 场景图标
- Banner图片
- 示例图片
- 背景素材

#### 4.7.2 素材上传

支持批量上传素材到云存储：
- 图片格式验证
- 自动压缩优化
- CDN加速分发

### 4.8 系统监控模块

#### 4.8.1 服务状态

监控各服务的运行状态：
- Core API服务状态
- AI服务状态
- 支付服务状态
- Redis服务状态

#### 4.8.2 性能监控

展示系统性能指标：
- API响应时间
- 请求成功率
- 错误日志

## 五、系统架构

### 5.1 前端架构

```
┌─────────────────────────────────────────┐
│           Vue 3 应用                     │
│  ┌─────────────────────────────────┐   │
│  │         视图层 (Views)           │   │
│  │  登录 | 仪表盘 | 用户 | 订单 | 场景 │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │        组件层 (Components)       │   │
│  │  表格 | 表单 | 图表 | 弹窗        │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │        状态管理 (Pinia)          │   │
│  │  用户状态 | 配置状态              │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │        API层 (Axios)             │   │
│  │  请求封装 | 拦截器 | 错误处理     │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 5.2 技术栈

- **前端框架**：Vue 3
- **UI组件库**：Element Plus
- **构建工具**：Vite
- **状态管理**：Pinia
- **HTTP客户端**：Axios
- **图表库**：ECharts

## 六、数据流程

### 6.1 管理员登录流程

```
输入用户名密码 → 发送登录请求 → 服务端验证
     ↓
验证通过 → 生成JWT令牌 → 返回前端
     ↓
前端保存令牌 → 跳转到仪表盘
```

### 6.2 场景配置流程

```
进入场景管理 → 选择场景 → 编辑配置
     ↓
配置步骤和选项 → 设置Prompt模板 → 保存配置
     ↓
配置生效 → 小程序端获取新配置
```
